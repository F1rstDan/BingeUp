# IndexedDB is the authoritative local user data store

所有必须共同备份、恢复和清除的本地用户数据统一以 IndexedDB 为唯一权威存储，包括长期设置、网站设置、安装引导状态、学习卡、复习日志和学习会话，以及未来由用户创建或导入的学习内容。`chrome.storage.local` 只保存可安全丢弃的临时运行状态，包括全局冷却、连续跳过次数和临时暂停截止时间；这些状态不得用于重建或修改学习进度。该选择以一次 IndexedDB 事务保证设置与学习进度不会形成半恢复状态；相比跨存储补偿回滚或 generation pointer/staging-swap，它更符合 Beta 阶段的最小完整方案。插件尚未公开发布，现有设置、学习进度和开发期备份均为可丢弃的测试数据，因此直接切换存储结构，不提供运行时数据迁移或开发期备份兼容代码；重新设计后的首次公开备份格式定为 v1，兼容承诺从该格式开始。
