# Triage Labels

技能以五个标准分流角色来表述。本文件把这些角色映射到本仓库 issue 追踪器中实际使用的标签字符串。

| Label in mattpocock/skills | Label in our tracker | 含义                              |
| -------------------------- | -------------------- | --------------------------------- |
| `needs-triage`             | `needs-triage`       | 维护者需要评估此 issue            |
| `needs-info`               | `needs-info`         | 等待报告者提供更多信息            |
| `ready-for-agent`          | `ready-for-agent`    | 已完整规格化，可供 AFK agent 接手 |
| `ready-for-human`          | `ready-for-human`    | 需要人工实现                      |
| `wontfix`                  | `wontfix`            | 不会处理                          |

当技能提到某个角色（如"应用 AFK-ready 分流标签"）时，使用上表中对应的标签字符串。

编辑右列以匹配你实际使用的词汇。
